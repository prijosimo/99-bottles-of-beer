# 99-bottles-of-beer

A simple web page that prints the full lyrics to the traditional song **"99 Bottles of Beer"**.  
Built with HTML, CSS, and JavaScript as part of my Web Design and Client Side Scripting course project.

---

## Features

- Prints the full lyrics from 99 down to 1  
- Clean, centered layout styled with CSS  
- JavaScript loop logic to generate verses dynamically  
- Easy to extend or adapt for similar counting songs  

---

## Technologies Used

- HTML for structure  
- CSS for styling  
- JavaScript for dynamic lyric generation
